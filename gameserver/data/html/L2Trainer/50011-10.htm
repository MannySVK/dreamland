Skill enchant is one of the way, how to improve your character by getting your skills better. Almost all skills are enchantable and there are various types of skill enchant.
Most common enchant types (trees) are listed below. Almost all enchantable skills have 2 main trees (e.g. Power, Cost).
Power - increases skill power Chance - increases skill land rate Cost - decreases skill cost Time - increases skill duration and other.
Skill enchanting is quite easy, it only requires Book of Giants. For each you are able to either enchant a skill (+1 Power -> +2 Power) or change the enchant type (+4 Cost -> +4 Power).
Both operations has certain chance to succeed, the higher enchant level is, the lower chance there is next enchant. If enchantment fails, the Book of Giants is still consumed but current skill enchant level is kept.
